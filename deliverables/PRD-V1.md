# 创新型中小企业成本管理系统 — 产品需求文档 (PRD)

> 版本：V1.0 | 日期：2026-06-12 | 编制：齐活林（交付总监）

---

## 一、产品概述

### 1.1 产品名称
**精益成本管理平台**（Lean Cost Management Platform）

### 1.2 产品定位
面向年产值 1000万～5亿元的成长型制造企业，提供**轻量化、智能化、可落地**的成本管理 SaaS 平台。将精益方法论固化为软件工具，实现"看得见、算得清、管得住、降得下"的成本管理闭环。

### 1.3 目标用户
| 用户角色 | 核心诉求 | 主要使用模块 |
|----------|----------|-------------|
| 企业老板/总经理 | 一眼看懂成本全貌 | 成本驾驶舱 |
| 财务经理 | 精细核算、多维度分析 | 成本核算、驾驶舱 |
| 采购主管 | 降低采购总成本 | 采购成本管理 |
| 生产主管 | 减少浪费、提升OEE | 生产精益模块 |
| 研发工程师 | 设计阶段控成本 | 研发设计成本 |
| 车间主任/班组长 | 库存控制、质量改善 | 库存控制、质量管理 |
| 全体员工 | 参与改善拿奖励 | 激励中心 |

### 1.4 核心价值主张
- **精益为核**：内置七大浪费识别、VSM、SMED、TPM等精益工具
- **全价值链覆盖**：研发→采购→生产→质量→库存→销售，形成闭环
- **数据驱动决策**：驾驶舱+异常预警+模拟测算
- **激励内嵌**：增量分享计算器+改善提案积分
- **轻量易用**：开箱即用，支持移动端

### 1.5 价格策略（参考）
| 版本 | 年费 | 适用规模 |
|------|------|----------|
| 基础版 | 6万元/年 | 小型企业（<5000万产值） |
| 专业版 | 18万元/年 | 中型企业（5000万-2亿） |
| 企业版 | 定制报价 | 大型企业（2亿-5亿） |

## 三、用户故事

### US-01 驾驶舱总览
**作为** 企业老板，**我想要** 登录后一眼看到所有关键成本指标，**以便** 快速判断经营健康度。

### US-02 多维成本核算
**作为** 财务经理，**我想要** 按订单/产品/工序三个维度查看完整成本构成，**以便** 精准定位成本超支环节。

### US-03 研发目标成本
**作为** 研发工程师，**我想要** 输入目标售价和利润后自动分解目标成本到BOM层级，**以便** 在设计阶段就控制成本。

### US-04 TCO采购决策
**作为** 采购主管，**我想要** 对比多个供应商的总拥有成本（含物流/质量/库存成本），**以便** 选出真正最优的供应商。

### US-05 OEE实时监控
**作为** 生产主管，**我想要** 在手机上录入停机和产量数据后自动计算OEE及六大损失，**以便** 及时发现设备效率问题。

### US-06 呆滞物料预警
**作为** 车间主任，**我想要** 系统自动识别超过90天未流动的物料并推送提醒，**以便** 尽快处理积压库存释放现金流。

### US-07 质量成本可视化
**作为** 质量经理，**我想要** 看到预防/鉴定/内部失败/外部失败四类成本的占比图，**以便** 决定质量投入方向。

### US-08 改善提案与奖励
**作为** 一线员工，**我想要** 提交改善提案后能看到评审进度和预期奖励积分，**以便** 积极参与降本增效活动。

## 四、需求池

### P0 — V1.0 必须实现（核心可用）

#### M01 - 成本驾驶舱
| 需求ID | 需求描述 | 验收标准 |
|--------|---------|---------|
| D-01 | 关键指标卡片：总成本、单位成本、成本费用率、库存周转率 | 实时显示，支持红黄绿灯状态 |
| D-02 | 成本结构环形图：材料/人工/制造费用/质量成本占比 | 点击可穿透到明细页 |
| D-03 | 单位成本趋势图：近12个月变化曲线 | 同比环比标注异常波动点 |
| D-04 | 异常预警列表：超预算/OEE下降/库存积压 | 至少5种预警规则 |
| D-05 | 改善项目动态进度展示 | 与激励中心联动 |

#### M02 - 全价值链成本核算
| 需求ID | 需求描述 | 验收标准 |
|--------|---------|---------|
| C-01 | 材料成本核算：按BOM归集材料消耗，支持标准vs实际对比 | 自动汇总BOM用量×单价 |
| C-02 | 人工成本核算：按工时费率计算 | 支持按班组/产品分摊 |
| C-03 | 制造费用分摊：ABC作业成本法引擎 | 支持自定义动因 |
| C-04 | 订单成本追溯：一键查询任意订单完整成本构成 | 穿透到采购单/工单/质检记录 |

#### M03 - 采购成本管理
| 需求ID | 需求描述 | 验收标准 |
|--------|---------|---------|
| P-01 | 供应商生命周期管理：准入→分级→QCDS考核→淘汰 | 自动生成雷达图 |
| P-02 | TCO采购决策：输入多维度成本自动计算TCO | 多供应商对比推荐 |
| P-03 | 采购价格趋势图 | 大宗商品指数联动预警 |
| P-04 | 集采建议：合并部门需求生成集采方案 | 降低采购频次 |

#### M04 - 库存成本控制
| 需求ID | 需求描述 | 验收标准 |
|--------|---------|---------|
| I-01 | ABC自动分类：按年耗用金额划分A/B/C类 | 可视化占比，支持自定义阈值 |
| I-02 | 安全库存计算器 | 支持多种模型（经典/动态） |
| I-03 | 呆滞物料预警 + 推送 | 原因分类标签 |
| I-04 | 库存周转模拟：调整参数看对周转率和现金流影响 | "如果-那么"情景分析 |

#### M05 - 激励中心
| 需求ID | 需求描述 | 验收标准 |
|--------|---------|---------|
| X-01 | 增量分享计算器：设定基线→算节约额→生成奖金 | 多种分享模型 |
| X-02 | 改善提案管理：提交→评审→实施→验收→奖励流程 | 积分排行榜可兑奖 |
| X-03 | 黄带/绿带/黑带认证管理 | 在线考试+证书 |

## 五、非功能需求

### 5.1 性能要求
- 页面加载时间 < 3秒
- 驾驶舱数据刷新 < 2秒
- 支持 50+ 用户并发使用

### 5.2 安全要求
- 数据传输 TLS 1.3 加密
- 敏感数据存储加密
- 多租户数据物理隔离
- RBAC 角色权限控制

### 5.3 兼容性
- PC端：Chrome / Edge / Firefox 最新两个版本
- 移动端：微信小程序 / 钉钉小程序
- 响应式布局适配平板

### 5.4 数据备份
- 每日自动备份，保留30天
- 支持一键导出数据

## 六、UI/UX 设计规范

### 6.1 设计风格
- **整体风格**：专业商务风 + 数据可视化优先
- **主色调**：深蓝 #1A73E8（科技/信任）+ 橙色 #FF9800（警示/行动）
- **背景色**：浅灰 #F5F7FA
- **卡片风格**：白色圆角卡片 + 微阴影
- **字体**：中文使用系统默认微软雅黑

### 6.2 布局框架
```
┌──────────┬────────────────────────────────┐
│          │        Header（搜索/通知/用户）   │
│  侧边栏  ├────────────────────────────────┤
│  导航菜单 │                                │
│          │         主内容区域               │
│  📊驾驶舱 │      （各模块页面内容）           │
│  💰核算   │                                │
│  🛒采购   │                                │
│  📦库存   │                                │
│  ⚙️生产   │                                │
│  ✅质量   │                                │
│  🏆激励   │                                │
│          │                                │
└──────────┴────────────────────────────────┘
```

### 6.3 各页面关键元素
- **驾驶舱首页**：4个KPI卡片 + 环形图 + 折线图 + 预警列表 + 项目进度条
- **列表页**：筛选条件栏 + 数据表格 + 分页 + 导出按钮
- **详情页**：Tab切换 + 关联数据卡片 + 操作按钮组
- **表单页**：步骤引导式填写 + 实时校验 + 保存草稿
